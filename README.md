# 📸 VeloxLedger (Smart Credit Ledger)

A sleek, modern business intelligence platform designed for local commerce. VeloxLedger integrates real-time facial recognition with a smart financial ledger to track debts and advances automatically.

## 🚀 The Architecture
1. **Edge AI:** Python/OpenCV for high-speed face detection.
2. **WebSockets:** Instant communication between your store camera and the web dashboard.
3. **Cloud Storage:** Image data is offloaded to Cloudinary for scalable, cloud-based deployment.
4. **Automation:** Node-Cron services for automated debt reminders.

## 🛠️ Tech Stack
* **Frontend:** React.js, Vite, Axios
* **Backend:** Node.js, Express.js, Socket.io
* **Database:** MongoDB Atlas
* **Cloud:** Cloudinary (Face Assets)

## 🚦 Deployment Ready
* **Backend:** Configured for Render (Web Service).
* **Frontend:** Configured for Netlify (Static Site).
* **Environment:** Fully containerized using `.env` variables for sensitive credentials.
